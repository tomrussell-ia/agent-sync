# Prettier Configuration

**Purpose**: Markdown formatting for consistency

**Files Formatted**: All `.md` files in the repository

## Settings

- Print width: 100 characters
- Prose wrap: preserve (don't reflow paragraphs)
- 2-space indentation

## Usage

```bash
# Install dependencies
npm install

# Format all markdown files
npm run format

# Check formatting without modifying
npm run format:check
```

## Pre-commit Hook

Prettier is configured as a pre-commit hook and will automatically format markdown files before commits.

## Manual Formatting

To format markdown manually:
```bash
npx prettier --write "**/*.md"
```

To check specific file:
```bash
npx prettier --check README.md
```
